# Simple Recipes
## Link
https://szymonborkowski.github.io/

## Introduction
This project is to help me learn html while also providing everyone with very simple recipes to follow and cook.

## Template
The following HTML template is currently used for all recipes:
```html
<!DOCTYPE html>
<html>
<body>

<h1>Recipe title</h1>

<hr>

<h2>Ingredients</h2>
<p>Required:</p>
<ul>
  <li>ingredient 1</li>
</ul>
<p>Optional:</p>
<ul>
  <li>ingredient 2</li>
</ul>

<hr>

<h2>Instructions</h2>
<ol>
  <li>instruction 1</li>
</ol>

<hr>

</body>
</html>
```
